This folder contains Python scripts and Jupyter notebooks used for data loading and transformation.

Main files:
- `data_cleaning.ipynb` — data quality checks and cleaning logic (Pandas)
- `load_data_to_mysql.ipynb` — automated loading of all CSV datasets into MySQL
- `mysql_connection_setup.py` — reusable connection helper
- `requirements.txt` — Python environment dependencies

The notebooks demonstrate practical ETL steps and data governance approaches.
