This folder contains all artifacts related to the database ER model of the Bookstore Database Project.

Contents:
- `bookstore_er_diagram.jpg` — the complete entity–relationship diagram
- `bookstore_schema.sql` — full SQL schema exported from dbdiagram.io
- `dbdiagram_export.dbml` — source model in DBML format

The model follows a normalized structure (3NF) and supports multi-genre books, warehouse-level inventory, and transactional order handling.
